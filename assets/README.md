# 🖼️ Assets Directory

This folder is used to store visual assets for the `Image reversal` skill's documentation and showcase.

## How to use placeholders

To make the showcase images appear correctly in the main `README.md`, simply upload your images into this folder and rename them to match the placeholder links:

- `original_sample.jpg` : Replace this with your original input image.
- `generated_sample.jpg` : Replace this with the AI-generated output image created using the extracted prompt.

You can also add more comparison pairs later by updating both this folder and the `README.md` file.
